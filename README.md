# Eduardo Cuducos

Source files of [cuducos.me](http://cuducos.me).

## Build instructions

Install [Node.js](http://nodejs.org) and [Bower](http://bower.io) dependencies: 

```console
$ npm install
$ bower install
```

Build the static website for _development_…

```console
$ grunt
```

… or _production_:

```console
$ grunt --env=production
```
